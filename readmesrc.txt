This describes the contents of the zip file.

build.xml 
-- the ant script that can be run to compile the program.

jep.jar 
-- a open source (as of 2.24) library that is used for the magnetic field calculations.

runbloch3d.bat
-- the batch file that is used to run the simulator as an application.  NOTE: you must define the java3d libarary path and the path to the required jar files to run the application.

runbloch3d.sh
-- the shell script file that is used to run the simulator as an application.  NOTE: you must define the java3d libarary path and the path to the required jar files to run the application.

src/**/*
-- all of the files that are compiled and used in building the application.
